## 데이터베이스를 이용한 인테리어 포트폴리오 사이트
> 프론트 앤드 개발부터 서버 사이트까지 통합한 포괄적 기능
> 데이터 입력 및 출력, 검색까지 전반적인 백앤드 기능 구현
> 어드민 기능으로 입력된 데이터의 검색, 수정 등 전반적인 관리
