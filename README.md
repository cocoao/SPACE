### 데이터베이스를 이용한 인테리어 포트폴리오 사이트
> 프론트 앤드 개발부터 서버 사이트까지 통합한 포괄적 기능<br>
> 데이터 입력 및 출력, 검색까지 전반적인 백앤드 기능 구현<br>
> Admin 기능으로 입력된 데이터의 검색, 수정 등 전반적인 관리<br>
> Q&A 게시판, Admin 페이지 탭 패널에 AJAX를 이용하여 비동기 데이터 출력
